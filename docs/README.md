## ✅ 기능구현 목록

- [x] 로또 구입 금액 입력 받기
    - [x] (예외) 입력이 숫자 인지 확인
    - [x] (예외) 1,000 단위로 나누어 떨이지는지 확인
- [x] 당첨 번호 받기
    - [x] `,`로 구분하는 메인 당첨 번호 받기
    - [x] (예외) 각 당첨 번호가 숫자인지 확인
    - [x] (예외) 로또 범위 초과하는지 확인
    - [x] (예외) 당첨 번호가 중복인지 확인
- [x] 보너스 번호 받기
    - [x] (예외) 로또 범위 초과하는지 확인
    - [x] (예외) 당첨 번호와 중복인지 확인
- [ ] 구매 수량에 맞는 당첨 번호 생성
- [ ] 사용자 번호랑 당첨 번호랑 비교하여 당첨 통계 출력
- [ ] 수익률 출력
